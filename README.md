DCGAN
Observations:
- GAN training is unstable and oscillatory
- Image quality does not monotonically improve
- Some generated samples remain poor due to uneven latent space learning
- No mode collapse observed in this experiment


WGAN
Experiments:
- DCGAN showed clear adversarial oscillation and unstable image quality.
- WGAN produced more stable training dynamics and slightly better image quality.
- WGAN reduced the subjective fluctuation of generated samples.
